# 云计算/容器/K8s 学习路径平台 - 海报文案

## 信息架构（文本描述）
- 顶部导航：首页 / 路线 / 专题 / 资源 / 我的
- 侧边菜单：与顶部同步，提供快捷跳转
- 首页：今日推荐、热门资源、整体进度
- 路线页：三阶段（基础 → Docker → K8s）卡片 + 完成度
- 专题页：搜索筛选、标签、分页，进入专题详情
- 资源页：类型/标签筛选，支持收藏，进入资源详情
- 详情页：Tabs 分栏（概览 / 章节 / 实验 / 相关资源）
- 我的：收藏抽屉、学习进度、最近浏览

## 设计亮点
- Ant Design 统一视觉，卡片+进度条突出学习路径
- Redux Toolkit + Context 保证收藏/主题/通知全局一致
- Express API + 本地 JSON 数据，前后端分离可直接运行
- 暗色/亮色一键切换，增强产品感
